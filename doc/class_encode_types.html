◆ UPCA_GS_1_DATABAR_COUPON

const UPCA_GS_1_DATABAR_COUPON = 63

Specifies that the data should be encoded with UPC coupon with GS1 DataBar addition barcode specification. An example of the input string: BarCodeBuilder->setCodetext("514141100906(8110)106141416543213500110000310123196000"), where UPCA part is "514141100906", DATABAR part is "(8110)106141416543213500110000310123196000". To change the caption, use barCodeBuilder->getCaptionAbove()->setText("company prefix + offer code");
